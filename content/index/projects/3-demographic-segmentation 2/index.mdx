**Customer & Workforce Segmentation**

Built unsupervised learning pipeline to cluster employees based on survey question responses
to verify employee archetypes.
- Built machine learning pipeline with SciPy/Scikit,Pandas, Numpy, Postgres
- Explored different unsupervised training models assessing correlation between
questions and groups
- Explored emergent cluster types, validated/presented accuracy, and logic with
stakeholders
- Performed demographic analysis resulting from hierarchical clustering and Self
Organising Maps
- Developed suite of Tableau dashboards providing insight into interactions
between employee clusters, survey questions and defining
clusters features
